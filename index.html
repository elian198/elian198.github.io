Challange Alkemy

Desafió de la plataforma Alkemy, es un Api Rest de Disney que nos permite crear Películas, Personajes asociados y géneros. También nos permite crear usuarios para la autentificación con Jwt, esa clave token se genera cuando el usuario se loguea correctamente, con ese token el usuario va poder navegar libremente por los end point de la Api.

Lenguajes utilizados Java, PostgressSql como bases de datos para almacenar los datos y Spring Boot como Framework de java.
